Gitlet

This is the final project I worked on from November to December in 2019 for CS61B: Data Structures at UC Berkeley. I used Java language with the Intellij as my IDE. The basic idea is to design a program that mimics Git called Gitlet because it cannot do everything that Git does. Starting from nothing the first step was to write up a design document that would follow a template where I came up with a strategy for what classes and data structures I would implement in my program. The next step was to begin programming in a test driven development (TDD) manner. I created unit tests and integration tests to confirm that the command that I just wrote code for worked. In this manner I designed the commands, init, add, rm, log, global-log, find, status, checkout, branch, rm-branch, reset, merge.

gitlet init: The purpose of this command is to initialize a git repository. This repository keeps track of commit objects, and a history of all previous snapshots of the directory that the repository has been initialized in. This also keeps track of references, such as what is the head pointing to at the moment.
