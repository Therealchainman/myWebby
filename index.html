Hall Effect with Semiconductor

In this lab assignment working with a partner, we investigated the physics of semiconductors and the Hall effect. The Hall effect was useful for discovering some properties of the semiconductor. The Hall effect is the name of the effect when you have a magnetic field perpendicular to an input current. This leads to a Lorentz force that deflects the charge carriers and develops a Hall voltage. The semiconductor is essentially a material with variable resistivity; which can act both like an insulator, at low temperatures, and a conductor, at high temperatures. You can dope a semiconductor to decrease its resistivity at low temperatures. We explored the following: determining the doping of the semiconductor from the Hall coefficient, investigating the disparity in the magnitude of the resistance and magnetoresistance, estimating the electron and hole concentrations, calculating the mobilities of the charged particles, determining the resistivity and conductivity as a function of temperature, and finding the temperature, 243.75K that corresponds to the transition between the extrinsic and intrinsic region of the semiconductor.

The analysis of the data was done in Jupyter Notebook with python. We found Pandas, Scipy, Numpy, and matplotlib libraries made the analysis easier.

Enigma

This project emphasized Object Oriented Programming (OOP) in the creation of an enigma machine in Java language that ran via the command line. Enigma is the encryption device used by the Germans in World War II that Alan turing worked in Bletchley Park to break. This required creating the machine, 3 versions of rotor (with inheritance), alphabet, and permutation classes. The program could be run by typing in the configuration, input, and output files. The configuration file would give all relevant information pertaining to the initialization of the machine with the number of rotor slots, movable rotors, alphabet, and all the potential rotors. I read in the configuration as a scanner and initialized the rotor objects necessary to insert them into my machine. The input file contained the settings line and then text that was to be encrypted or decrypted. The encrypted or decrypted text would be saved to an output file. If no output file was entered then it would just print the text.
